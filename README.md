## 有啥功能🧐
- 首页：查看当天天气的概览
- 数据：查看当天天气的详细数据
- 小时：查看后12小时，每小时天气的概览（*非真实数据，仅展示* ）
- 一周：查看后7天，每天天气的概览
- 初次打开App自动定位城市
- 更改城市：选择热门城市，或搜索城市名
- 通知栏显示当天天气的概览
- 设置：更换背景：  
  可选择不同天气展示不同的视频背景  
  还可选择自定义图片作背景（*仅展示* ）
